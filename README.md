# 🏙️ Civic Issue Reporting & Resolution System

A web-based platform that allows citizens to report civic issues (like potholes, garbage, water leakage, street light problems) and enables authorities to track, manage, and resolve them efficiently.

---

## 🚀 Features

- 📢 Report civic issues with description and location
- 📷 Upload images for better clarity
- 📍 Location-based issue tracking
- 🔄 Real-time status updates (Pending → In Progress → Resolved)
- 👨‍💼 Admin dashboard for authorities
- 🔍 Filter and search issues
- 🧾 User authentication (Login/Signup)

---
